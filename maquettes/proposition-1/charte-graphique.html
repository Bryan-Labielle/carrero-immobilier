05 — Animations & Effets

Principes

  • Reveal au scroll : translateY(40px) + opacity, cubic-bezier(0.16, 1, 0.3, 1)
  • Stagger delay : 0.12s entre chaque élément
  • Cartes : translateY(-8px) + scale image 1.05 au hover
  • Boutons primaires : fill de gauche à droite (scaleX)
  • Lignes qui s'allongent au hover (w-8 → w-14)
  • Texture grain fixe en overlay (opacity 0.03)

Valeurs CSS

easing: cubic-bezier(0.16, 1, 0.3, 1)

reveal: opacity 1s + transform 1s

hover-colors: duration-300

hover-transform: duration-500

btn-fill: scaleX 0.5s

/* Grain texture */

feTurbulence baseFrequency="0.9"

opacity: 0.03 · position: fixed

/* Dot pattern (sections sombres) */

radial-gradient 1px · size 32px

opacity: 0.02-0.04

07 — Règles de design

DO

  • Fond crème #F8F6F1 (jamais blanc pur pour le body)
  • Vert #4A5A4F en accent ET pour les labels de section (primary-500, pas 400)
  • DM Serif Display pour les titres d'impact (+ italic pour emphase)
  • JetBrains Mono pour labels de section (font-medium), prix inline, refs
  • Méta données en neutral-500 (#737370) minimum, jamais neutral-400
  • Boutons rectangulaires sans arrondi
  • Cartes en format portrait 3/4
  • Grille de sections séparées par 1px (gap-px)
  • Numérotation des sections (01, 02, 03...)
  • Texture grain en overlay global
  • Dot pattern sur les fonds sombres

DON'T

  • Pas de couleurs vives ou saturées
  • Pas de border-radius (tout est rectangulaire)
  • Pas d'ombres lourdes (shadow-sm maximum)
  • Pas d'animations rapides ou flashy
  • Pas d'Inter, Roboto, Arial, system fonts
  • Pas de gradients colorés
  • Pas de fond blanc pur pour le body
  • Pas de texte centré par défaut (sauf CTA)
  • Pas de bordures opacity 100% (toujours /40 ou /60)
  • Pas de texte en neutral-400 pour du contenu lisible (réserver aux placeholders)
  • Pas de font-light sur les méta données (réserver au body text)
